# OutLinear

## Requirements

* A running [PostgreSQL installation](https://www.postgresql.org/download/) with a database named "outlinear"
* A `config.ts` located in the `server` directory with a configuration matching the example

## Run Instructions

* Start PostgreSQL
* Create a new database with the name `outlinear` (or the name corresponding to the database field of your `config.ts` file) using the command `createdb <database-name>`
* Open two terminals
* In the first terminal, navigate to the `client` directory
* Run `yarn` to install the dependencies
* Run `yarn start` to bundle the code with webpack for static serving
* In the second terminal, navigate to the `server` directory
* Run `yarn` to install the dependencies
* Run `yarn start` to start the server, which will establish a connection to the database
* Then, use a browser to navigate to `localhost:3000`, wherein the project will be visible
